### Getting Started with frontend

Clone the project

```bash
git clone https://github.com/ZombiePander/R_W_APP.git
```

Go to the project directory

```bash
cd R_W_APP
```

Install dependencies.

```bash
npm install
```

Start a local dev server for the app

```bash
expo start
```
